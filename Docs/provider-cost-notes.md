# TTS / 声音克隆费用说明

更新时间：2026-02-07（估算）

## 结论

- 纯本地离线（iOS 系统语音）几乎无 API 成本，但“像本人”能力有限。
- 远程克隆音色（方案 A）能显著提升拟真度，但会产生按字符或按分钟计费。
- 建议首周先跑小流量验证，再根据日均时长切换到更合适套餐。

## 粗略估算方法

- 成人语速约 150 字/分钟（中文可按 120~180 字/分钟浮动）
- 每天朗读 1 小时，月总时长约 1800 分钟
- 可折算为约 162 万字符/月（按英文字母计价时为近似值）

## 常见供应商（示例）

- ElevenLabs：通常按分钟/额度分层，超额单价随套餐变化。
- Google Cloud TTS：常见按每百万字符计费，不同模型价格不同。

## 项目建议

- MVP 阶段：
  - 默认使用远程克隆，失败自动回退本地离线。
  - 对每本书做分段缓存，避免重复合成。
- 费用控制：
  - 设置每日字符上限。
  - 优先缓存命中。
  - 对长文先提示预计消耗。
